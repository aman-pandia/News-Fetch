# News-Fetch
Fetch current headlines from rss feed of TOI

## Requirements
```shell
pip3 install requests
pip3 install beautifulsoup4
```

## Usage 
```shell
git clone https://github.com/aman-pandia/News-Fetch.git
cd News-Fetch
python3 rss_toi.py
```
